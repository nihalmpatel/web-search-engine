Implementation Report for Namespaces in XML 1.1 Implementation Report for Namespaces in XML 1.1 XML Core Working Group 30 October 2003 Abstract This document lists implementation feedback about Namespaces in XML 1.1 implementations known by the XML Core Working Group at this time. There are various tests for Namespaces 1.1--along with other XML tests--available via the XML Test Suite page. RXP "The latest test version of RXP supports (...) Namespaces 1.1" See the RXP page. XML Apache Xerces-C++ Xerces-C++ version 2.3.0 includes "a full experimental implementation of Namespaces in XML 1.1, W3C Candidate Recommendation of 18 December 2002." See the list of features. XML Apache Xerces2 Java Xerces2 Java version 2.4.0 "handles namespaces according to the XML Namespaces 1.1 Candidate Recommendation." See the release page.
